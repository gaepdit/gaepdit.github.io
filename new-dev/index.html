New Developer Setup

Steps to get set up as a new developer with EPD IT.

Required software

Install the following software:

  • Visual Studio IDE
  • SQL Server Management Studio (SSMS)
  • Git CLI
  • KDiff3

Required Git setup

Setting up Git correctly is very important. Do not skip this step!

Configure Git by running each of the following commands at the command prompt:

  • git config --global user.name "Your Name"
  • git config --global user.email your.email@dnr.ga.gov

Add the user .gitignore file to your user profile directory (%userprofile%).

Required online accounts

  • Create an Atlassian account. Provide your manager with your user name so you can be added to the GA EPD-IT JIRA and Bitbucket teams.

Other recommended software

The following can be installed if or when needed:

  • Notepad++ (Full-featured text editor)
  • Visual Studio Code (Text editor and IDE)
  • Axosoft GitKraken (Git GUI - get license from manager)
  • Sourcetree (Another Git GUI)
  • Node.js and NPM (JavaScript runtime and package manager)
  • Pandoc (Document format converter)
  • Inkscape (Vector graphics editor)
  • Greenshot (Screenshot tool)
  • ApexSQL Refactor (SSMS plugin for formatting SQL)
  • ApexSQL Refactor settings file (for use with above)

Optional online services

  • Create a Google account for Google Docs. You can use your DNR email address to create the account if desired.
  • Create a Gravatar account and add an avatar for your DNR email address.
